##  Conceptos básicos de HTML y CSS

###  HTML – Estructura y etiquetas básicas

**Etiqueta raíz**
- `<html></html>` → Todo lo que está dentro hace parte de la página web.

**Cabecera**
- `<head></head>` → Contiene la información de la página (no visible).
- `<title></title>` → Título de la página.
- `<meta charset>` → Define la codificación de caracteres.
- `<meta name>` → Información adicional no visible.

**Cuerpo**
- `<body></body>` → Todo lo visible de la página web.

**Texto**
- `<p></p>` → Párrafos.
- `<h1>…<h6>` → Títulos y subtítulos.

**Enlaces**
- `<a href="direccion"></a>` → Enlaces para redireccionar.
- `target="_self"` → Abre el enlace en la misma pestaña (valor por defecto).
- `target="_blank"` → Abre el enlace en una nueva pestaña.
- `title="descripcion"` → Muestra una descripción al pasar el cursor.

**Imágenes**
- `<img src="ruta">` → Inserta una imagen.
- `alt="descripcion"` → Texto alternativo si la imagen no carga.

**Rutas**
- Rutas relativas y absolutas (dependen de la ubicación del archivo).

**Inputs**
- `<input type="">` → Campos de entrada de datos.
- Atributos importantes:
  - `required` → Campo obligatorio
  - `name` → Identificador del input
  - `placeholder` → Texto de ayuda
  - `value` → Valor enviado al servidor

---

###  Introducción a CSS

- **Estilos en línea** → Estilos aplicados directamente a un elemento.
- **Clases** → Permiten asignar un nombre a los elementos para aplicar estilos.

**Propiedades básicas**
- `font-family` → Tipografía
- `font-size` → Tamaño del texto
- `font-weight` → Grosor del texto

**Box Model**
Describe cómo se diseñan y distribuyen los elementos:
- **Content** → Contenido (texto o imágenes)
- **Padding** → Espacio interno entre contenido y borde
- **Border** → Borde que rodea el contenido
- **Margin** → Espacio externo entre elementos
